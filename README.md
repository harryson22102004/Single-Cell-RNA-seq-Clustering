# Single-Cell RNA-seq Clustering
Computational pipeline for analyzing high-dimensional transcriptomic data to identify cell populations.

### 🔬 Research Highlights
- **Dimensionality Reduction**: Implements PCA or t-SNE logic to handle the high dimensionality of scRNA-seq datasets.
- **Unsupervised Learning**: Uses K-Means or Louvain-style clustering to discover distinct cell types from gene expression matrices.
- **Bioinformatics Focus**: Addresses technical noise and sparsity in genomic data.

**Tech Stack:** Python, `Scikit-Learn`, `Scanpy`.
